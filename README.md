# elastic-regions

Elastic Cloud Regions — interactive map component

Single index.html file, no dependencies, drop it anywhere and it works.

What it does:

- World map with all 27 Elastic Cloud regions marked (AWS, GCP, Azure)
- Tabs to fly to Americas / EMEA / APAC
- Click a region → zooms in; drag + scroll to explore
- Sidebar list with provider grouping, serverless badge, filters
- Hover tooltip with region ID and providers


What's easy to add later:

- Links to deploy pages per region
- Real-time availability status per region
- Mobile touch support (pinch-zoom, collapsible sidebar)
- More regions as they launch (just add an entry to the REGIONS array)
